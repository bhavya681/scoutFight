# FighterOrg Design System

## Brand Personality

FighterOrg should feel:

* Professional
* Trustworthy
* Premium
* Athletic
* Modern

Avoid:

* Gaming-style UI
* Excessive animations
* Visual clutter
* Neon cyberpunk aesthetics

---

## Visual Hierarchy

Priority order:

1. Primary CTA
2. Fighter identity
3. Key statistics
4. Supporting information
5. Secondary actions

Every page should have one obvious primary action.

---

## Typography

### Headings

* Large
* Bold
* High contrast

### Body Text

* Easy to scan
* Short paragraphs
* Consistent spacing

### Labels

* Compact
* Clear
* Secondary emphasis

---

## Buttons

### Primary

* Red gradient
* High contrast
* Main conversion action

Examples:

* Sign Up
* Get Verified
* Contact Fighter

### Secondary

* Dark background
* Lower emphasis

Examples:

* Learn More
* View Details

### Border Radius

12px

---

## Cards

* 12–16px radius
* Consistent padding
* Subtle borders
* Strong hover states

Used for:

* Fighter profiles
* Events
* Organizations

---

## Spacing

Use 8px grid system.

Examples:

* 8px
* 16px
* 24px
* 32px
* 48px
* 64px

Avoid random spacing values.

---

## Mobile First

Requirements:

* Touch targets ≥ 44px
* Responsive layouts
* No horizontal scrolling
* Important actions visible above the fold

---

## Accessibility

* Sufficient color contrast
* Keyboard navigation
* Clear focus states
* Semantic HTML

---

## Trust Signals

Important for FighterOrg:

* Verification badges
* Fighter records
* Promotion logos
* Professional imagery
* Clear statistics

Trust should be visible immediately.

---

## Conversion Principles

Every page should answer:

1. What is this?
2. Why should I care?
3. What should I do next?

Primary CTA should always be obvious.

---

## UI Review Checklist

When auditing UI:

* Visual hierarchy
* Consistency
* Mobile experience
* Accessibility
* Trust
* Conversion
* Performance perception
