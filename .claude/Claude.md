# ScoutFight

Combat sports talent discovery platform.

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind
- Supabase

## Coding Rules

- Strict TypeScript
- No any
- Server Actions preferred
- Mobile first
- Accessibility considered

## UI Principles

- Clean visual hierarchy
- Minimal clutter
- Fast loading
- Professional sports aesthetic

## Security Rules

- RLS required
- Validate all inputs
- No secrets in frontend

## Deployment

- Vercel frontend
- Supabase backend
